Persiapan Wawancara

Contoh pertanyaan wawancara: Bagaimana cara membuat fungsi sederhana di JavaScript?

}

JavaScript & HTML DOM Objects

JavaScript Objects

Objek dalam JavaScript

Contoh pembuatan objek dalam JavaScript:

// Contoh objek
let mobil = {
    merk: "Toyota",
    warna: "Merah",
    nyalakan: function() {
        console.log("Mobil menyala!");
    }
};
mobil.nyalakan();

HTML DOM Objects

Objek HTML DOM

Contoh manipulasi elemen HTML dengan JavaScript:

// Contoh manipulasi DOM
document.body.style.backgroundColor = "#f0f8ff";
console.log("Background halaman telah diubah!");
