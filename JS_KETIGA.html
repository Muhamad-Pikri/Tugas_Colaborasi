JavaScript Rangkuman

Function Definitions

Menentukan fungsi dalam JavaScript dengan kata kunci function.


function sapa() {
    console.log("Halo, Dunia!");
}
sapa();

Function Parameters

Nilai yang diterima fungsi ketika dipanggil.


function tambah(a, b) {
    return a + b;
}
console.log(tambah(5, 3));

Function Invocation

Proses pemanggilan atau eksekusi fungsi.


function cetakPesan() {
    console.log("Fungsi telah dipanggil.");
}
cetakPesan();

Function Call

Memanggil fungsi untuk mengeksekusi kode di dalamnya.


let nama = "John";
function sapa(nama) {
    console.log("Halo, " + nama);
}
sapa(nama);

Function Apply / Bind

Menetapkan konteks pada fungsi menggunakan apply() dan bind().


let orang = {
    nama: "John",
    sapa: function(kota) {
        return "Halo, " + this.nama + " dari " + kota;
    }
};
console.log(orang.sapa.apply({nama: "Jane"}, ["Jakarta"]));

Function Closures

Fungsi yang memiliki akses ke variabel dari lingkup luar meskipun eksekusi selesai.


function buatCounter() {
    let hitung = 0;
    return function() {
        hitung++;
        return hitung;
    };
}
let counter = buatCounter();
console.log(counter());
console.log(counter());

Class Intro

Pengantar untuk kelas dalam JavaScript untuk membuat objek.


class Orang {
    constructor(nama) {
        this.nama = nama;
    }
    sapa() {
        return "Halo, nama saya " + this.nama;
    }
}
let orangBaru = new Orang("John");
console.log(orangBaru.sapa());

Class Inheritance

Pewarisan properti dan metode antar kelas.


class Hewan {
    constructor(nama) {
        this.nama = nama;
    }
    sapa() {
        return "Saya adalah " + this.nama;
    }
}
class Kucing extends Hewan {
    meong() {
        return "Meong!";
    }
}
let kucingBaru = new Kucing("Kucing Persia");
console.log(kucingBaru.sapa());
console.log(kucingBaru.meong());

Class Static

Metode dan properti statis yang tidak memerlukan instansi kelas.


class Matematika {
    static tambah(a, b) {
        return a + b;
    }
}
console.log(Matematika.tambah(10, 20));

JS Callbacks

Fungsi yang diteruskan sebagai argumen untuk dipanggil kembali pada waktu tertentu.


function prosesData(callback) {
    console.log("Memproses data...");
    callback();
}
prosesData(function() {
    console.log("Data selesai diproses!");
});

JS Asynchronous

Pengolahan kode tanpa memblokir eksekusi lainnya, menggunakan callback atau promise.


console.log("Mulai");
setTimeout(() => {
    console.log("Proses Asynchronous");
}, 2000);
console.log("Selesai");

JS Promises

Objek yang mewakili nilai yang mungkin belum tersedia tetapi akan tersedia di masa depan.


let janji = new Promise((resolve, reject) => {
    setTimeout(() => resolve("Berhasil!"), 2000);
});
janji.then((hasil) => console.log(hasil));

JS Async/Await

Penggunaan async dan await untuk menangani operasi asinkron lebih mudah.


async function fetchData() {
    let data = await new Promise((resolve) =>
        setTimeout(() => resolve("Data diterima"), 2000)
    );
    console.log(data);
}
fetchData();

DOM Intro

Pengantar tentang Document Object Model (DOM) untuk mengakses dan memanipulasi elemen HTML.


document.getElementById("judul").textContent = "DOM Example";

DOM Methods

Metode untuk berinteraksi dengan elemen DOM, seperti getElementById().


let elemen = document.getElementById("judul");
elemen.style.color = "blue"; // Mengubah warna teks elemen dengan ID 'judul'

DOM Document

Metode yang digunakan untuk mengakses objek dokumen HTML.


console.log(document.title); // Menampilkan judul dokumen
document.title = "Judul Baru"; // Mengubah judul dokumen

DOM Elements

Elemen-elemen HTML yang dapat dimanipulasi melalui JavaScript.


let elemen = document.getElementById("judul");
elemen.style.color = "blue"; // Mengubah warna teks elemen dengan ID 'judul'

DOM HTML

Interaksi antara JavaScript dan elemen-elemen HTML dalam DOM.


document.getElementById("konten").innerHTML = "Isi baru ditambahkan!"; // Mengubah konten HTML

DOM Forms

Penggunaan JavaScript untuk berinteraksi dengan elemen formulir HTML.


function submitForm() {
    let nama = document.forms["myForm"]["nama"].value; // Mengambil nilai dari elemen form
    alert("Nama Anda: " + nama); // Menampilkan alert dengan nilai dari form
}

DOM CSS

Penggunaan JavaScript untuk memanipulasi gaya CSS elemen DOM.


let elemen = document.getElementById("box");
elemen.style.backgroundColor = "yellow"; // Mengubah warna latar belakang
elemen.style.width = "200px"; // Mengubah lebar elemen
elemen.style.height = "100px"; // Mengubah tinggi elemen

DOM Animations

Penggunaan JavaScript untuk membuat animasi pada elemen DOM.


let box = document.getElementById("box");
let pos = 0;
function animasi() {
    if (pos === 350) return;
    pos++;
    box.style.left = pos + "px"; // Menggerakkan elemen ke kanan
    requestAnimationFrame(animasi); // Memanggil animasi berikutnya
}
animasi();

DOM Events

Peristiwa yang terjadi pada elemen DOM, seperti klik atau input.


document.getElementById("tombol").onclick = function() {
    alert("Tombol diklik!"); // Menampilkan alert ketika tombol diklik
};

DOM Event Listener

Menambahkan fungsi untuk menangani event pada elemen DOM.


let tombol = document.getElementById("tombol");
tombol.addEventListener("click", function() {
    alert("Event listener berhasil!"); // Menampilkan alert ketika tombol diklik
});

DOM Navigation

Menavigasi antar elemen dalam struktur DOM.


let parent = document.getElementById("list");
let firstChild = parent.firstElementChild; // Mengambil elemen pertama
let lastChild = parent.lastElementChild; // Mengambil elemen terakhir
console.log(firstChild.textContent); // Menampilkan konten elemen pertama
console.log(lastChild.textContent); // Menampilkan konten elemen terakhir

DOM Nodes

Objek dalam DOM yang mewakili elemen atau teks.


let node = document.createElement("p"); // Membuat elemen paragraf baru
node.textContent = "Paragraf baru."; // Menambahkan teks pada paragraf baru
document.body.appendChild(node); // Menambahkan elemen paragraf ke body

DOM Collections

Daftar objek yang dikelola oleh DOM, seperti node list.


let koleksi = document.getElementsByClassName("item"); // Mengambil koleksi elemen dengan class 'item'
Array.from(koleksi).forEach((elemen) => console.log(elemen.textContent)); // Menampilkan konten dari setiap elemen dalam koleksi

Daftar koleksi node dalam dokumen HTML yang bisa diakses dan dimanipulasi.


let nodeList = document.querySelectorAll("li"); // Mengambil semua elemen
