JavaScript Rangkuman Lanjutan

JS Object Methods

Metode adalah fungsi yang terkait dengan objek, misalnya toString() atau valueOf().

const obj = { name: "John", age: 30 };
console.log(obj.toString()); // Outputs: [object Object]

JS Object Display

Menampilkan objek menggunakan console.log(), atau mengubahnya menjadi string.

const person = { name: "John", age: 30 };
console.log(person); // Outputs: { name: 'John', age: 30 }

JS Object Constructors

Gunakan konstruktor seperti function untuk membuat objek baru dengan properti yang sama.

function Person(name, age) {
  this.name = name;
  this.age = age;
}
const person1 = new Person("John", 30);
console.log(person1); // Outputs: Person { name: 'John', age: 30 }

JS Strings

String adalah teks yang dideklarasikan dengan tanda kutip ganda atau tunggal.

let greeting = "Hello, world!";
console.log(greeting); // Outputs: Hello, world!

JS String Methods

Metode string seperti length, slice(), dan replace() digunakan untuk memanipulasi teks.

let message = "Hello, JavaScript!";
console.log(message.length); // Outputs: 19
console.log(message.slice(0, 5)); // Outputs: Hello
console.log(message.replace("JavaScript", "World")); // Outputs: Hello, World!

JS String Search

Mencari string menggunakan metode seperti indexOf() atau search().

let str = "I love JavaScript!";
console.log(str.indexOf("JavaScript")); // Outputs: 7
console.log(str.search("love")); // Outputs: 2

JS String Templates

Gunakan template literal (``) untuk interpolasi string dengan ${expression}.

let name = "John";
let message = Hello, ${name}!;
console.log(message); // Outputs: Hello, John!

JS Numbers

Angka adalah tipe data untuk bilangan, mendukung operasi aritmatika.

let num1 = 10;
let num2 = 5;
console.log(num1 + num2); // Outputs: 15

JS BigInt

Tipe data untuk angka yang sangat besar, dibuat dengan menambahkan n di akhir angka.

let bigNumber = 1234567890123456789012345678901234567890n;
console.log(bigNumber); // Outputs: 1234567890123456789012345678901234567890n

JS Number Methods

Metode angka seperti toFixed() dan toString() memformat angka.

let num = 123.456;
console.log(num.toFixed(2)); // Outputs: 123.46
console.log(num.toString()); // Outputs: 123.456

JS Number Properties

Properti bawaan seperti MAX_VALUE dan MIN_VALUE.

console.log(Number.MAX_VALUE); // Outputs: 1.7976931348623157e+308
console.log(Number.MIN_VALUE); // Outputs: 5e-324

JS Arrays

Array adalah daftar nilai, dideklarasikan dengan tanda kurung siku.

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits); // Outputs: ['Apple', 'Banana', 'Cherry']

JS Array Methods

Metode array seperti push(), pop(), sort(), dan filter().

let fruits = ["Apple", "Banana", "Cherry"];
fruits.push("Orange");
console.log(fruits); // Outputs: ['Apple', 'Banana', 'Cherry', 'Orange']

fruits.pop();
console.log(fruits); // Outputs: ['Apple', 'Banana', 'Cherry']

JS Array Search

Mencari elemen dalam array dengan indexOf(), find(), atau includes().

let numbers = [1, 2, 3, 4, 5];
console.log(numbers.indexOf(3)); // Outputs: 2
console.log(numbers.includes(6)); // Outputs: false
console.log(numbers.find(num => num > 3)); // Outputs: 4

JS Arrow Function

Simplifikasi sintaks fungsi dengan tanda panah (=>).

const add = (a, b) => a + b;
console.log(add(2, 3)); // Outputs: 5

JS Classes

Definisi dan penggunaan kelas untuk membuat objek.

class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
  greet() {
    console.log(Hello, ${this.name}!);
  }
}
const person1 = new Person("John", 30);
person1.greet(); // Outputs: Hello, John!

JS Modules

Fitur untuk membagi kode menjadi modul-modul yang dapat diimpor dan diekspor.

Untuk menggunakan modul, pastikan kode disertakan dalam file terpisah dan menggunakan import dan export. 
export function greet(name) {
  return Hello, ${name}!;
}

import { greet } from './greeting.js';
console.log(greet("John")); // Outputs: Hello, John!

JS JSON

Format data untuk pertukaran informasi, mendukung struktur objek dalam JavaScript.

let jsonData = '{"name": "John", "age": 30}';
let obj = JSON.parse(jsonData);
console.log(obj.name); // Outputs: John

JS Mistakes

Kesalahan umum dalam JavaScript dan cara menghindarinya.

Contoh kesalahan: mengubah variabel dalam forEach() bisa menyebabkan hasil yang tidak terduga. 
let nums = [1, 2, 3];
nums.forEach((num) => num = num * 2); // This does not change the original array
console.log(nums); // Outputs: [1, 2, 3]

Object Get / Set

Penggunaan get dan set untuk mengakses dan mengubah properti objek.

class Person {
    constructor(name, age) {
        this._name = name;
        this._age = age;
    }

    get name() {
        return this._name;
    }

    set name(value) {
        this._name = value;
    }

    get age() {
        return this._age;
    }

    set age(value) {
        this._age = value;
    }
}

const person = new Person("Alice", 25);
console.log(person.name);  // Outputs: Alice
person.name = "Bob";
console.log(person.name);  // Outputs: Bob

Object Protection

Melindungi properti objek menggunakan getter, setter, atau pengaturan akses.

class BankAccount {
    constructor(balance) {
        this._balance = balance;
    }

    get balance() {
        return this._balance;
    }

    set balance(amount) {
        if(amount >= 0) {
            this._balance = amount;
        } else {
            console.log("Balance cannot be negative!");
        }
    }
}

const account = new BankAccount(100);
console.log(account.balance);  // Outputs: 100
account.balance = 50;          // Valid
console.log(account.balance);  // Outputs: 50
account.balance = -10;         // Outputs: Balance cannot be negative!
