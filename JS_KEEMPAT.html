Rangkuman JavaScript & JSON

JS Window

Menangani jendela browser seperti ukuran dan posisi.

Contoh:

alert("Ukuran jendela browser: " + window.innerWidth + " x " + window.innerHeight);

JS Screen

Informasi tentang layar pengguna, seperti lebar dan tinggi layar.

Contoh:

alert("Lebar layar: " + screen.width + " px, Tinggi layar: " + screen.height + " px");

JS Location

Memberikan informasi atau mengubah URL halaman saat ini.

Contoh:

window.location.href = "https://www.google.com";

JS History

Metode untuk berinteraksi dengan riwayat penjelajahan browser.

Contoh:

history.back(); // Kembali ke halaman sebelumnya

JS Navigator

Memberikan informasi tentang browser dan sistem operasi pengguna.

Contoh:

alert("Nama browser: " + navigator.appName);

JS Popup Alert

Menampilkan kotak pesan pop-up menggunakan alert().

Contoh:

alert("Ini adalah pesan pop-up!");

JS Timing

Fungsi untuk menangani penundaan, seperti setTimeout() dan setInterval().

Contoh:

setTimeout(function() { alert("2 detik sudah berlalu!"); }, 2000);

JS Cookies

Metode untuk mengakses dan mengelola cookies di browser.

Contoh:

document.cookie = "username=John Doe";

Web API Intro

Pengantar ke API Web untuk komunikasi antara browser dan server.

Contoh:

fetch('https://api.example.com/data') .then(response => response.json()) .then(data => console.log(data));

Web Forms API

Interaksi dengan elemen formulir di halaman web.

Contoh:

document.getElementById("myForm").submit();

Web Storage API

Menawarkan penyimpanan data di sisi klien melalui LocalStorage dan SessionStorage.

Contoh:

localStorage.setItem("username", "Jane Doe"); alert(localStorage.getItem("username"));

AJAX Intro

Metode untuk melakukan request asinkron tanpa memuat ulang halaman.

Contoh:

const xhr = new XMLHttpRequest(); xhr.open("GET", "https://api.example.com/data", true); xhr.onload = function() { if (xhr.status == 200) { console.log(xhr.responseText); } }; xhr.send();

JSON Parse

Menangani parsing string JSON menjadi objek JavaScript dengan JSON.parse().

Contoh:

const jsonString = '{"name":"John", "age":30}'; const jsonObj = JSON.parse(jsonString); alert(jsonObj.name); // John

JSON Stringify

Mengubah objek JavaScript menjadi string JSON menggunakan JSON.stringify().

Contoh:

const obj = {name: "John", age: 30}; const jsonStr = JSON.stringify(obj); alert(jsonStr);

jQuery Selectors

Selektor jQuery untuk memilih elemen HTML dan berinteraksi dengannya.

Contoh:

$(document).ready(function(){ $("p").click(function(){ alert("Paragraf diklik!"); }); });

jQuery HTML

Manipulasi konten HTML menggunakan jQuery.

Contoh:

$("p").html("Konten HTML telah diubah!");

jQuery CSS

Pengaturan gaya CSS menggunakan jQuery.

Contoh:

$("p").css("color", "blue");

JS Canvas

Elemen <canvas> di HTML5 untuk menggambar grafik dan gambar dinamis menggunakan JavaScript.

Contoh:

const canvas = document.getElementById("myCanvas"); const ctx = canvas.get const ctx = canvas.getContext("2d"); ctx.fillStyle = "green"; ctx.fillRect(50, 50, 150, 100); // Menggambar kotak hijau

JS Plotly

Pustaka JavaScript untuk membuat grafik interaktif dengan Plotly.

Contoh:

var trace1 = { x: [1, 2, 3, 4], y: [10, 11, 12, 13], type: 'scatter' }; var data = [trace1]; Plotly.newPlot('myDiv', data);

JS Chart.js

Pustaka JavaScript untuk membuat grafik menggunakan Chart.js.

Contoh:

var ctx = document.getElementById('myChart').getContext('2d'); var chart = new Chart(ctx, { type: 'bar', data: { labels: ['Red', 'Blue', 'Yellow', 'Green', 'Purple', 'Orange'], datasets: [{ label: 'Jumlah', data: [12, 19, 3, 5, 2, 3], backgroundColor: ['red', 'blue', 'yellow', 'green', 'purple', 'orange'], }] } });

JS Google Chart

Pustaka untuk menampilkan grafik menggunakan Google Chart API.

Contoh:

google.charts.load('current', {'packages':['corechart', 'bar']}); google.charts.setOnLoadCallback(drawStuff); function drawStuff() { var data = new google.visualization.arrayToDataTable([ ['City', '2010 Population', '2000 Population'], ['New York City', 8175133, 8008278], ['Los Angeles', 3792621, 3694820], ['Chicago', 2695598, 2896016], ]); var chart = new google.visualization.BarChart(document.getElementById('chart_div')); chart.draw(data, {width: 600, height: 400}); }

JS D3.js

Pustaka JavaScript untuk membuat grafik dan visualisasi data dinamis menggunakan D3.js.

Contoh:

var dataset = [25, 30, 45, 60, 20]; var svg = d3.select("body").append("svg") .attr("width", 200) .attr("height", 200); svg.selectAll("circle") .data(dataset) .enter().append("circle") .attr("cx", function(d, i) { return i * 50 + 25; }) .attr("cy", 100) .attr("r", function(d) { return d; }) .style("fill", "blue");

JS Exercises

Latihan-latihan untuk melatih keterampilan JavaScript.

Contoh:

// Coba buat program untuk menghitung faktorial function factorial(n) { if (n === 0 || n === 1) return 1; return n * factorial(n - 1); } console.log(factorial(5)); // Output: 120
