JS Home

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web lebih interaktif.

Contoh:

document.body.innerHTML = "Halo, Dunia!";

JS Introduction

Memperkenalkan JavaScript sebagai bahasa pemrograman untuk web dan non-web, seperti server.

Contoh:

console.log("Belajar JavaScript Dasar");

JS Output

Menghasilkan output menggunakan metode seperti document.write(), alert(), atau console.log().

Contoh:

console.log("Output di Konsol");
document.write("Output di Halaman Web");
alert("Output berupa Notifikasi");

JS Statements

Instruksi JavaScript dieksekusi satu per satu, dipisahkan dengan titik koma.

Contoh:

let a = 5;
let b = 10;
console.log(a + b);

JS Syntax

Sintaks menentukan cara JavaScript ditulis, seperti menggunakan variabel, fungsi, dan pernyataan.

Contoh:

function sapa() {
    console.log("Halo, JavaScript!");
}
sapa();

JS Comments

Komentar dibuat dengan // untuk satu baris atau /* */ untuk beberapa baris.

Contoh:

// Ini adalah komentar satu baris
/*
Ini adalah komentar 
beberapa baris
*/

JS Variables

Variabel dideklarasikan menggunakan var, let, atau const.

Contoh:

let nama = "John";
const umur = 30;
console.log(nama, umur);

JS Let

let digunakan untuk mendeklarasikan variabel dengan lingkup blok.

Contoh:

let x = 10;
{
    let x = 20;
    console.log(x); // 20
}
console.log(x); // 10

JS Const

const digunakan untuk mendeklarasikan variabel yang nilainya tidak bisa diubah.

Contoh:

const pi = 3.14;
console.log(pi);

JS Operators

Menyediakan operator seperti aritmatika, perbandingan, dan logika.

Contoh:

let a = 5;
let b = 3;
console.log(a + b); // 8

JS Arithmetic

Menggunakan operator seperti +, -, *, dan /.

Contoh:

let hasil = 10 * 2 / 5;
console.log(hasil);

JS Assignment

Operator penugasan seperti =, +=, dan *= digunakan untuk menetapkan nilai.

Contoh:

let x = 5;
x += 3;
console.log(x); // 8

JS Data Types

Data tipe utama adalah string, number, boolean, object, dan undefined.

Contoh:

let nama = "John"; // string
let umur = 30; // number
let isMarried = false; // boolean

JS Functions

Fungsi dideklarasikan menggunakan function, digunakan untuk menjalankan kode tertentu.

Contoh:

function tambah(a, b) {
    return a + b;
}
console.log(tambah(5, 10));

JS Objects

Objek adalah kumpulan properti yang mendefinisikan karakteristiknya.

Contoh:

let mobil = {
    merk: "Toyota",
    tahun: 2020
};
console.log(mobil);

JS Object Properties

Properti adalah nilai yang terkait dengan objek, diakses menggunakan dot atau bracket notation.

Contoh:

let buku = {
    judul: "JavaScript Dasar",
    halaman: 200
};
console.log(buku.judul);
